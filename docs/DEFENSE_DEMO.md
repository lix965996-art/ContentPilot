# 8–12 分钟答辩演示流程

1. 双击启动脚本，说明依赖只在首次安装，打开登录页（30 秒）；
2. 用 operator 登录，展示工作台真实业务摘要（30 秒）；
3. 内容管理新建一篇原文（45 秒）；
4. AI 工作室选择三平台并生成，展示 Prompt 版本、耗时、质量、编辑比例和审核（90 秒）；
5. 配图推荐展示中英文关键词、本地降级和署名信息（45 秒）；
6. 时间推荐展示曲线、贡献分、置信度和备选时段（60 秒）；
7. 日历新建排期并拖拽；进入发布中心执行 Mock 发布并查看幂等日志（90 秒）；
8. 数据复盘下载模板、导入样例、展示平台/时间组图表和摘要（90 秒）；
9. 实验管理展示分组、样本和汇总结果（60 秒）；
10. admin 查看配置脱敏、用户角色和审计日志（45 秒）；
11. 强调 MOCK/SIMULATED 边界与真实数据采集计划（30 秒）。

演示前运行全量测试并保留 `docs/TEST_REPORT.md`；外部网络不可用时保持默认 Mock 模式。
